

# netflixPark

기능구현의 발전을 목표로 전세계적으로 성공적인 스트리밍 서비스 중 대표적 사례인 넷플릭스를 모방했습니다

#역할분배

박청원: 로그인, 좋아요기능 ,서치 , 동적화면 출력 

윤영민: 회원가입 , 프로필

강동언: 비디오 업로드 , 멤버쉽
